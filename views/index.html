API Project: Timestamp Microservice

User Stories :

  1. If the date string is empty the service uses the current timestamp.
  2. If the date string is valid the api returns a JSON having the structure
    {"unix": 1479663089000 ,"utc": "Sun, 20 Nov 2016 17:31:29 GMT"}.
  3. If the date string is invalid the api returns a JSON having the structure
    {"error": "Invalid Date" }.

Example Usage:

  • https://scrawny-foxglove.glitch.me/api/timestamp/2015-12-25
  • https://scrawny-foxglove.glitch.me/api/timestamp/1450137600

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
